# Sistema de Folgas e Férias - Mercado Livre

Sistema web para controle de folgas e férias dos colaboradores da BRSP15.

## Funcionalidades

- ✅ Solicitação de folgas (1-7 dias)
- ✅ Solicitação de férias (5-30 dias)
- ✅ Seleção de gestor (Eliane ou Gabriel)
- ✅ Calendário interativo com feriados
- ✅ Validação de limites (máx. 3 colaboradores por dia)
- ✅ Edição e exclusão de solicitações pendentes
- ✅ Busca por matrícula
- ✅ Responsivo para mobile

## Como usar

1. Acesse o sistema no GitHub Pages
2. Preencha seu matrícula e nome
3. Selecione o gestor responsável
4. Escolha o tipo de solicitação (Folga ou Férias)
5. Selecione as datas
6. Clique em "Enviar Solicitação"

## Observações

- O sistema armazena dados localmente no navegador
- Dados são preservados entre sessões
- Compatível com todos os navegadores modernos

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API
